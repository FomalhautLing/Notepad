# Notepad

[![release](https://img.shields.io/github/v/release/FomalhautLing/Notepad)](https://github.com/FomalhautLing/Notepad/releases/latest)

仿自 Windows11 Notepad 的轻量级文本编辑器，以满足偶尔查看文本文件内容的需要
